<h2>Application Competition for Intelligent Reality (ACIR) of <br><a href="https://icir.ieee.org" target=_blank>IEEE International Conference on Intelligent Reality (ICIR)</a></h2>
<H3>Overview</H3>
The Application Contest for Intelligent Reality (ACIR) challenges developers to create AI machine learning <I>Intelligent Reality</I> (IR) applications to provide realtime decision-making user experience with analytics capability for the immersive (VR/AR/MR/XR) environment.
<H3>Eligibility </H3>
The ACIR Competition is open to anyone interested in Intelligent Reality development â€“ researchers, university students, hobbyists, professionals, or anyone else. 

<H3>Teams </H3>
You may register as a team (up to five per team) or an individual. 

<H3>Instructions</H3>
Competition participants must meet the following guidelines: 
<UL>
<LI>Student must attend an accredited university (for students only)
<LI>Include AI machine learning techniques and run within VR/AR/MR/XR
<LI>Be able to run on a cross-platform emulator for evaluation
<LI>All implementation must be based on the original work.
  <LI>Grant IEEE to post codes and data (if any) under public IEEE-ICIR-ACIR GitHub
  </UL>
  <H3>Evaluation criteria</H3>
  <OL>
<LI>Quality of idea: creativity and originality
<LI>Implementation of idea: user experience, design, and performance
<LI>Potential impact on Intelligent Reality: deploy AI machine learning in VR/AR/MR/XR
  </OL>
  <H3>Awards</H3>
Winners (at the discretion of the Evaluation Team)
<UL>
  <LI>1st Place: up to $1000 with 100 Ranking Points and IEEE certificate
<LI>2nd Place: up to $500 with 50 Ranking Points and IEEE certificate
<LI>3rd Place: up to $300 with 30 Ranking Points and IEEE certificate
<LI>All other submitters will receive 10 Ranking Points for participation
  </UL>
  <b>Ranking Points: Accumulated points submitter earned through all competitions </b>

<H3>Competition submission instructions</H3>
Participating teams must submit competition application via <a href="https://docs.google.com/forms/d/1igIJIIobhfGW8YaHFL2_LthQtgTELPsVYbpdG121SBE/viewform?ts=5fc004ff&edit_requested=true">ACIR online system</a> with answers to the following questions:
<OL>
<LI>Team Name
<LI>Authors(s) (one author per line, maximum five authors)<br>
  <I>Last name, First name, Email address, Organization/School, Country</I>
<LI>Application Description
<LI>How does AI machine learning enhance your IR application?
<LI>What AI machine learning technique(s) do you plan to implement?
<LI>What operating systems (PC/Mac/etc.) do you plan to use?
<LI>What development platform and tools/libraries do you plan to implement?
  </OL>
For any inquiries, please send emails to icir@ieee.org.
